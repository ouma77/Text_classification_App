Comment ça marche ?
Afin de simplifier les processus et d’automatiser les actions pour gagner en efficacité sur le traitement des dossiers, ROUX Afrique Network vous permet d'automatiser votre inventaire et faciliter la tâche pour les experts avec cette application basé sur l'intelligence artificielle.
Cette application permet de classifier les équipements industriels selon leurs désignations en quelques étapes simples :
  • Importer un fichier excel qui comporte la colonne de Designation
  • Appliquer le classifieur ML en cliquant sur le bouton classifié
  • Téléchargez votre nouveau fichier excel et voila!
